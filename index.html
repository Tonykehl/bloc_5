API de suggestion de prix pour les propriétaires de voitures

/predict

Méthode : POST

Entrée requise : Les données JSON doivent être envoyées avec une clé "input" contenant une liste des données d'entrée.

Sortie attendue : La réponse JSON contient une clé "prediction" contenant une liste des prédictions de prix.

Exemple d'utilisation :

        {
          "input": [[7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8], [7.0, 0.27, 0.36, 20.7, 0.045, 45.0, 170.0, 1.001, 3.0, 0.45, 8.8]]
        }
